# Special Tools Available To You

## The Oracle

- **What it is:** Strategic technical advisor for architecture decisions, deep code analysis, and hard debugging.
- **How to use it:** When the user asks you to “use the oracle,” run `copilot -p "<your prompt>" --agent oracleAgent --model gpt-5.2`.
- **When to use:** Complex design choices, post-implementation self-review, multi-system tradeoffs, or after 2+ failed fix attempts.
- **Avoid for:** Simple file edits, first-pass fixes, or trivial questions answerable from context.
