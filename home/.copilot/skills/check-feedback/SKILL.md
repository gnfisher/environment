---
name: check-feedback
description: Fetch comments and reviews for the branch's github pull request
direct-model-invocation: true
---

Use gh or github mcp tools to fetch comments and reviews for the branch's github
pull request. Consider the feedback in the context of the changes and the larger
codebase. Determine if the feedback is valid and actionable, and if so, what
changes should be made to address it, and share your concise clear opinion.

If the user includes "yolo" in the request just address the feedback you find valid.
